Overview

This session focuses on publishing applications to the Google Play Store. This is a recorded session which walks you through the publishing process and Google's development portal.

Objectives

  1. Configure you app for publishing
  2. Create a Google Play developer account
  3. Create an app package
  4. Submit your application for publishing

Lecture

AND220 - Publishing an Android app to the Google Play Store
